# ГдеСдать (Project T) ♻️

**ГдеСдать** — это современная эко-платформа для поиска и управления пунктами приема вторсырья. Проект включает в себя интерактивную карту, систему отзывов и полноценный личный кабинет с возможностью модерации точек.

## ✨ Основные функции

* 🗺️ **Интерактивная карта:** Поиск ближайших пунктов приема на базе Yandex Maps API с фильтрацией по категориям отходов.
* 👤 **Личный кабинет:** Управление профилем, заполнение контактных данных (город, телефон, описание) и загрузка аватара.
* 🔒 **Безопасность:** Авторизация на базе JWT и функционал смены пароля с проверкой старого значения.
* 💬 **Отзывы и реакции:** Возможность оставлять комментарии и ставить лайки/дизлайки пунктам приема.
* 🛡️ **Панель модератора:** Интерфейс для добавления новых точек и актуализации цен на вторсырье.
* 🔔 **Уведомления:** Система информирования пользователей о статусе их заявок и новых акциях.

## 🛠 Технологический стек

### Backend (Python/Django)
* **Django 6.0.3** — основной каркас системы.
* **Django REST Framework 3.17.1** — реализация API.
* **SimpleJWT & Djoser** — работа с пользователями и токенами.
* **PostgreSQL + PostGIS** — хранение геоданных.
* **Pillow** — обработка изображений профиля.
* **Geopy** — геокодирование и работа с координатами.

### Frontend (React)
* **React (Vite)** — быстрый и отзывчивый интерфейс.
* **Yandex Maps API** — картографический функционал.
* **Axios** — взаимодействие с бэкендом.
* **Bootstrap 5** — сетка и стилизация компонентов.

## 🚀 Быстрый запуск (Docker)

Проект полностью контейнеризирован. Для запуска выполните следующие шаги:

1.  **Клонируйте репозиторий:**
    ```bash
    git clone [https://github.com/Snowdy54/project_t.git](https://github.com/Snowdy54/project_t.git)
    cd project_t
    ```

2.  **Соберите и запустите контейнеры:**
    ```bash
    docker-compose up -d --build
    ```

3.  **Примените миграции базы данных:**
    ```bash
    docker-compose exec backend python manage.py migrate
    ```

4.  **Создайте суперпользователя (для доступа в админку):**
    ```bash
    docker-compose exec backend python manage.py createsuperuser
    ```

### Доступ к сервисам:
* **Frontend:** [http://localhost:5173](http://localhost:5173)
* **Backend API:** [http://localhost:8000/api/](http://localhost:8000/api/)
* **Панель администратора:** [http://localhost:8000/admin/](http://localhost:8000/admin/)
